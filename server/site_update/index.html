What is Safe Alarm?

This project includes a webpage and mobile application which controls fire alarms. The fire alarms can either be activated or deactivated based on the user's commands. The alarm is connected to wifi which enables controls form the user. When the alarm rings, the user will get a notification on his/her mobile device. After that, he/she can either choose to call the police if there is actually an emergency or he/she can choose to turn it off if there is no emergency.
